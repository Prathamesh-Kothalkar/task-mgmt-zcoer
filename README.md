# Task Management System 
A modern Task Management System developed for ZEAL Institute of Engineering & Research, under the guidance of the Head of the IT Department.
This project aims to streamline task allocation, tracking, and management within the institute using a clean and scalable web architecture.
## Features
- Notifications via emails and in-app
- Credentials Generation and send to Users
- Forgot Password (Reset Mandatory)
- Security i.e Account locked after hit number of incorrect credentials
- Scalable backend setup using Node.js
- Developer-friendly project structure

## Pre-requist
Make sure the following are installed on your system:
- Node.js (Latest LTS recommended)
- pnpm (Fast, disk-efficient package manager)
# Installation

Use the package manager [pnpm]

```bash
npm i pnpm
```
